Welcome to the Plone Collective.

This is an organization for developers of Plone add-ons and related software.

Join

To join the Plone Collective, please open a ticket. Note: you must have a GitHub account to open a ticket. You can create one for free here.

New repository

To create a new repository or fork an existing repository to the collective, please tell us the desired name of your new repository or give us the URL to fork an existing repository.

Other issues

For all other issues, please open a ticket.
